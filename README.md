# Comanda Digital - Restaurante do Jânio

Aplicação web em React para controle de pedidos (comanda digital) do **Restaurante do Jânio**, em Cuiabá - MT.

## Funcionalidades

- Cardápio organizado por categorias (pratos principais, porções, sobremesas, bebidas e cervejas).
- Controle de quantidade de cada item da comanda.
- Cálculo automático do valor total da mesa.
- Geração de código **PIX copia e cola** com valor exato da conta.
- Envio do pedido diretamente para o **WhatsApp** do restaurante.
- Link rápido para localização no **Google Maps**.

## Tecnologias utilizadas

- React
- lucide-react (ícones)
- Tailwind CSS (classes de estilo utilitárias)
